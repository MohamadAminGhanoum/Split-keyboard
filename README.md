# Split-keyboard
This project is a custom split keyboard that I have customly built. I have built this keyboard because I feel that i am weak at working with 3d designing, and therefore I would like to use this as an opportunity for me to strengthen my 3d designing knowledge. Aside from that, i want this to hopefully replace my current old mechanical keyboard, and this would honestly make my table look a lot cleaner, as it is wireless.
